Prueba GIT
==========
PruebaGIT es un proyecto resultado de una práctica de la asignatura *Entornos de Desarrollo* sobre GIT, EGIT y [GitHub](http://www.github.com). Su objetivo es crear un flujo de trabajo básico que permita iniciar nuevos proyectos con GIT, gestionarlos y subirlos a GitHub.

Instalación
-----------
Para la correcta instalación y funcionamiento del proyecto harán falta al menos:

* Git.
* Eclipse.
* El plugin EGIT para Eclipse.
* Una cuenta en GitHub.
